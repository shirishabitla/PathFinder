# 🚀 PathFinder

An AI-powered Career Recommendation System built using **Python, Flask, Machine Learning, and Ollama LLM**.

## 📌 Project Overview

PathFinder helps users discover suitable career paths based on their:

- Skills
- Qualification
- Experience

The system uses a hybrid approach:
- 📊 Dataset-based recommendation
- 🤖 LLM-based recommendation (Ollama - Qwen2.5:3B)

## ✨ Features

- Career Recommendation
- Skill Extraction
- Feature Accuracy Calculation
- Missing Skills Detection
- Personalized Learning Roadmap
- Learning Resources (YouTube, Coursera, GeeksforGeeks)
- AI Chatbot Support
- Offline LLM Support using Ollama

## 🛠️ Technologies Used

- Python
- Flask
- HTML
- Machine Learning
- TF-IDF
- Cosine Similarity
- RapidFuzz
- Ollama
- Qwen2.5:3B

## ▶️ How to Run

1. Install Python
2. Install the required packages

```
pip install -r requirements.txt
```

3. Start Ollama

```
ollama run qwen2.5:3b
```

4. Run the application

```
python app.py
```

5. Open the browser

```
http://127.0.0.1:5000
```
